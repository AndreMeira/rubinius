What is Rubinius?

A virtual machine for running Ruby programs and a Ruby core library.

Get Started >

Why does it matter?

For Businesses

We aim to increase Ruby use and help Ruby applications toward a unified compatibility. This means for you: less fragmentation in developer knowledge, not getting backed into hardware vendor lock-in.

For Developers

A spec driven Ruby implementation. A core library written in Ruby. More first class objects than you can shake a stick at. Super easy to contribute. Low barrier to entry. If you can breathe, you can commit.
