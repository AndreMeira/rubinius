What is Rubinius?

Rubinius is an implementation of the Ruby programming language.

The Rubinius bytecode virtual machine is written in C++. The bytecode compiler is written in pure Ruby. The vast majority of the core library is also written in Ruby, with some supporting primitives that interact with the VM directly.

Rubinius uses a precise, generational garbage collector. Rubinius provides a foreign function interface (FFI) and a compatible C-API for C/C++ extensions written for the standard Ruby interpreter (often referred to as MRI—Matz's Ruby Implementation).
